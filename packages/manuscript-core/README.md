# @anointed-pages/manuscript-core

Canonical manuscript schema and core data structures.

## Structure

- **Front Matter** - Title page, copyright, dedication, TOC, preface
- **Chapters** - Main body content
- **Back Matter** - Appendix, index, glossary, bibliography, about the author

## Principles

- Structure first, layout second
- No inline formatting
- Chapter-based organization
- Style references only (no embedded styles)
